Package gql.dsl

Package gql.dsl Description

This package contains internal classes to build GraphQL schemas via Groovy DSL
  • Interface Summary 
    Interface Description
    ScalarsAware This interface is just a container for basic Scalar types defined in graphql-java Scalars class.
  • Trait Summary 
    Trait Description
    ListAware Any builder implementing this trait will be capable of converting any GraphQLType to a GraphQLList
    NonNullAware Any builder implementing this trait will be capable of converting any GraphQLType to a GraphQLNonNull, meaning that that type is mandatory
  • Class Summary 
    Class Description
    ArgumentBuilder This builder holds information to build a new argument, an instance of GraphQLArgument
    DynamicFieldCapture It's used as a delegate object for a closure in order to capture all properties called within that Closure.
    EnumTypeBuilder Builds a GraphQLEnumType
    EnumTypeBuilder.Entry Represents every enum type key/value pair
    InputTypeBuilder Builds a new GraphQLInputObjectType
    InputTypeBuilder.FieldBuilder Builds an instance of GraphQLInputObjectField
    ObjectTypeBuilder Builds a new GraphQLObjectType
    ObjectTypeBuilder.FieldBuilder Builder that makes possible to create a field of type GraphQLFieldDefinition
    ObjectTypeBuilder.InterfacesBuilder Builder to aggregate a given type object's interfaces
    QueryBuilder Builder aimed to create a compliant GraphQL queryString using a static typed DSL
    QueryBuilder.ReturnsBlockBuilder @since 0.1.0
    ScalarTypeBuilder Builds a new GraphQLScalarType.
    SchemaBuilder Builds a new GraphQLSchema
    SchemaMergerBuilder Builder responsible to merge several GraphQL IDL definitions into a single GraphQLSchema instance and make possible to map those definitions to their correspondent DataFetcher instances afterwards
    SchemaMergerBuilder.SchemaMappingBuilder This type defines the relationships between the types defined in the IDL documents and the data fetchers responsible to fetch the data
    SchemaMergerBuilder.SchemaMappingFieldBuilder This class has functions to map type fields to fetchers.
