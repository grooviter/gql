Package: gql.dsl

[Groovy] Class SchemaMergerBuilder

  • Inheritance:
  • java.lang.Object
  • gql.dsl.SchemaMergerBuilder

  • Builder responsible to merge several GraphQL IDL definitions into a single GraphQLSchema instance and make possible to map those definitions to their correspondent DataFetcher instances afterwards
