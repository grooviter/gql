Package: gql.dsl

[Groovy] Class ScalarTypeBuilder

  • Inheritance:
  • java.lang.Object
  • gql.dsl.ScalarTypeBuilder

  • Builds a new GraphQLScalarType. Apart from the name and description, a scalar type is made out of three functions:

    • serialize: serializes data coming from a fetcher result to a valid output
    • parseValue: converts an incoming query input to a specific type
    • parseLiteral: converts an incoming literal to a specific type
    Since:
    0.1.3
