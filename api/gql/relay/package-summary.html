Package gql.relay

Package gql.relay Description

This package contains internal classes to build Relay schemas using a Groovy DSL
  • Interface Summary 
    Interface Description
    RelayPrimitivesAware Basic primitives of a Relay implementation
  • Class Summary 
    Class Description
    ConnectionBuilder A Connection is a way of manipulating one-to-many relationships easy, using a standardized way of expressing these one-to-many relationships.
    ConnectionFieldBuilder Builds a new field of type connection.
    NodeBuilder A `Node` type is just an instance of GraphQLObjectType that must have an identifier.
    RelaySchemaBuilder Builds a new GraphQL schema but adding to the schema context the basic Relay primitives
