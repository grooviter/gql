      • static DataFetcher<?> errorFetcher(InstrumentationFieldFetchParameters parameters, @DelegatesTo(GraphQLErrorBuilder) Closure options)

        When instrumenting a DataFetcher via Instrumentation.instrumentDataFetcher you should return whether the current data fetcher with no changes or a modified data fetcher. Sometimes you may want to return an error as a result of a given condition. This method creates an instance of a DataFetcher which sets a GraphQLError to the current execution context and returns no data

        Returns:
        an instance of DataFetcher
        Parameters:
